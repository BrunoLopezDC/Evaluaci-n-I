# Ecosistema de Monitoreo Canino

Sistema integral diseñado para llevar el registro médico y el monitoreo de las rutinas de mascotas pequeñas o rescatadas, esta información se sincroniza de forma automática en diferentes pantallas para que el tutor siempre tenga el control del bienestar del perrito.

## Pantallas y Dispositivos
* Smartphone: Funciona como el carnet médico principal y permite registrar las comidas, vacunas y paseos.
* Smartwatch: Sirve para contar los pasos de la mascota durante los paseos y activar alarmas locales sin necesidad de internet.
* Smart TV: Funciona como un tablero de información en la sala de la casa, mostrando un resumen del día de la mascota de manera visual.

## Herramientas de Desarrollo
* Desarrollo de las aplicaciones: Flutter.
* Base de Datos y Nube: Supabase con PostgreSQL.
* Conectividad: Conexión en tiempo real para que la televisión se actualice sola y Bluetooth para conectar el reloj con el celular.
